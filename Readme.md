# Mini Desktop OS

A browser based simulation of a desktop operating system built using
**HTML, CSS and Vanilla JavaScript**. This project recreates core
interactions of a modern desktop environment such as a taskbar, start
menu, draggable windows, and right click context menus.

The goal of this project was to practice **DOM manipulation, UI layout
design, and event driven programming** while building a realistic user
interface similar to Windows.

------------------------------------------------------------------------

## Live Features

This project includes several core desktop operating system
interactions.

### Desktop Interface

Desktop icons placed on a grid layout similar to real operating systems.

Users can double click icons to open applications.

### Window System

Applications open inside movable windows.

Each window includes

Title bar\
Close button\
Drag functionality

Windows can be repositioned freely across the desktop.

### Start Menu

A Windows inspired Start Menu built with a blurred glass effect.

Includes

Search bar\
Application grid\
User profile section\
Power options

### Taskbar

The bottom taskbar acts as the main system navigation.

Includes icons for

Chrome\
Edge\
GitHub\
Discord\
Xbox

Clicking an icon opens the associated website.

### Power Menu

Power button inside the Start Menu provides system actions.

Options include

Sleep\
Restart\
Shut down

### Right Click Context Menu

The desktop includes a custom context menu that appears on right click.

Menu includes options such as

View\
Sort by\
Refresh\
New folder\
Display settings

Submenus expand similar to the Windows interface.
